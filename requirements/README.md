# audition-project
Uses flickr's public feed to retrive recently uploaded cat photos.

# requirements
* node.js v6.10.3

# getting started
1. ```git clone https://github.com/emilyrenee/audition-project.git```
2. ```cd requirements```
2. Run ```npm install```
3. Run ```node script.js```

To view the data retrived,```cat result.json```
